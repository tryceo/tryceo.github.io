# Portfolio Website

Static portfolio site designed to work directly on GitHub Pages.

## Files

- `index.html` contains the page structure and copy.
- `styles.css` contains the visual design and responsive layout.
- `script.js` adds scroll reveals and the footer year.
- `.nojekyll` tells GitHub Pages to serve the site as plain static files.

## Deploy to GitHub Pages

1. Create a new GitHub repository and push these files to the default branch.
2. In GitHub, open `Settings` -> `Pages`.
3. Under `Build and deployment`, choose `Deploy from a branch`.
4. Select your default branch and the `/ (root)` folder.
5. Save, then wait for GitHub Pages to publish the site.

## Before Publishing

- Replace the placeholder contact links in `index.html`.
- Update the page title and copy with your preferred name and branding.
- Add your own domain later if you want a custom URL.
